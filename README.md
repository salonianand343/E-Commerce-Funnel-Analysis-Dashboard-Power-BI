# 🚀 E-Commerce Funnel Analysis Dashboard | Power BI

## 📊 Project Overview
This project presents an interactive **Funnel Analysis Dashboard** built using **Power BI** to analyze user behavior across an e-commerce platform.  

It tracks the complete customer journey — from **product views → cart → purchase** — helping identify key drop-off points and opportunities to improve conversion rates.

---

## 🎯 Objective
- Analyze the complete user funnel (**View → Cart → Purchase**)  
- Identify major drop-off stages  
- Understand category-wise performance  
- Provide actionable business insights  

---

## 📌 Key Metrics
- 📉 **Overall Conversion Rate:** 6.24%  
- 🛒 **Purchase Sessions:** 15K  
- 👀 **View Sessions:** 235K  
- ⚠️ **Cart Abandonment Rate:** 21.66%  
- 💰 **Total Revenue:** 310M+  

---

## 🔍 Insights & Findings
- Significant drop-off observed between **View → Cart stage**  
- Cart abandonment is relatively lower, indicating users who add items are likely to purchase  
- **Electronics (Smartphones)** category dominates both revenue and conversions  
- High brand concentration suggests potential dependency risk  

---

## 💡 Recommendations
- Improve product listing pages (images, descriptions, reviews)  
- Run **A/B testing** to optimize conversion rates  
- Focus marketing efforts on high-performing categories  
- Reduce dependency on a single brand by diversifying offerings
